# Yoloo

Yoloo is a social portal which allows travelers ask questions such as accommodation, transportation, currency, food, budget, and places to visit. If you are travelling solo or in a group, even if you are just dreaming about travelling the whole world, Yoloo is a great place to start your adventure.
